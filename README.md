#Responsive Portfolio Homepage
This project incoorporates a responsive grid system, custom breakpoints, and responsive images to display a PDF design mockup in HTML.

##How to run
Simply clone the repository and open index.html in any web browser

##Compatability
This project incooporates the <picture> element which is not compatible with Internet Explorer. Though, it is compatible with Edge.

##APIs and Frameworks
This project incooporates a responsive grid system using Bootstrap. It also uses placeholder images hosted by Placehold.it
